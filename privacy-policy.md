# Privacy Policy for BetterYT

Last updated: February 2026

BetterYT is a Chrome extension designed to let users create custom YouTube feeds by organizing channels into categories. Your privacy is important, and this extension is built to work without collecting or transmitting personal data.

## Data Collection

BetterYT does not collect, store, or transmit any personal information.

The extension does not collect:
- Personally identifiable information
- Financial or payment information
- Authentication information
- Personal communications
- Location data
- Web browsing history outside of YouTube
- Analytics or tracking data

## Local Storage

BetterYT uses Chrome's local storage API to store:
- User-created categories
- Lists of YouTube channels added to those categories
- Extension preferences

All of this data is stored **locally on your device** and never sent to external servers.

## Website Access

BetterYT runs on YouTube pages in order to:
- Display custom category chips on the homepage
- Allow users to add channels to categories
- Generate curated feeds from selected channels

The extension only interacts with content on YouTube and does not monitor activity on other websites.

## Data Sharing

BetterYT does not:
- Sell data
- Share data
- Transfer data to third parties
- Send data to external servers

All functionality operates entirely within the user's browser.

## Open Source

BetterYT is an open-source project. The source code can be reviewed to verify how the extension works and how data is handled.
